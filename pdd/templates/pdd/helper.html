HELPER

Запустив программу, можно увидеть текущую дату и ВК автора

Дата окончания активации

Инструкция по запуску

  • Для начала нужно включить диспетчер задач(CTRL + ALT + ESC) поверх всех окон
  • Далее нажать CTRL + ALT + DEL и выбрать диспетчер задач
  • В диспетчере задач в верхней менюшке выбрать "файл"
  • Запустить новую задачу
  • Обзор
  • Слева в проводнике выбрать флешку(PDD HELPER)
  • Открыть папку Pdd_helper
  • Открыть файл Pdd_helper.exe

    • Инструкция по программе

  • Программа сама будет нажимать правильную цифру ответа, твоя задача только нажимать ENTER
  • Если программа не дала ответ в течение 15сек. значит она не знает ответ

    • Что значит дата до окончания активации?

    Программа работает только до даты которая указана в программе, после окончания активации, программа соответственно работать не будет.

    Программа не работает, что делать?

    Если по каким-то причинам программа перестала работать, необходимо написать сюда.
